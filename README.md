# EPS-DTI
Integrating Expert Priors with Dual-Level Contrastive Repre-sentation Learning for Data-Efficient Drug Discovery
# Explanation
This project is the code for the paper "Integrating Expert Priors with Dual-Level Contrastive Repre-sentation Learning for Data-Efficient Drug Discovery" submitted to the *IEEE/ACM Transactions on Computational Biology and Bioinformatics*. We have uploaded some of the data and code; the remaining code will be uploaded in full after the paper is accepted.
